# Лабораторна робота №7: Computer Vision та багатопотоковість у C++

##Опис
Програма виконує захоплення відео з камери та дозволяє накладати різні фільтри (Інверсія, Blur, Canny, Sepia, Sobel). Головною особливістю є режим *FACE* (вмикається клавішею `F`), який виконує детекцію облич за допомогою нейромережі ResNet-10 (cv::dnn). 
Щоб уникнути блокування інтерфейсу обчисленнями нейромережі, детекція винесена в окремий фоновий потік. Для демонстрації асинхронності у фоновий потік додано штучну затримку в 500 мс - при цьому саме відео залишається плавним (30 FPS, бо це максимум камери).

##Інструкція
Крок 1. Встановлення залежностей цією послідовністю команд:
chmod +x preinstall.sh
./preinstall.sh

Крок 2. Компіляція:
chmod +x build.sh
./build.sh

Крок 3. Запуск програми:
chmod +x run.sh
./run.sh

##Керування програмою
Гарячі клавіші:
F - Режим детекції обличчя
1 - Нормальний режим (без фільтрів)
2 - Інверсія кольорів (інверсія)
3 - Gaussian Blur (розмиття)
4 - Canny фільтр (виділення меж)
5 - Ефект Сепії
6 - Фільтр Собеля 
Esc - Вихід з програми

## Архітектура (Взаємодія класів)
Програма побудована так:
*CameraProvider*: Відповідає за ініціалізацію камери (з вибором індексу) та отримання кадрів (getFrame). Встановлює формат (MJPEG) для забезпечення високого FPS.
*FaceDetector*: Клас, що реалізує фоновий Worker Thread за допомогою `std::thread`. Використовує `std::mutex` для безпечного отримання копії кадру від основного потоку та повернення масиву знайдених облич (`cv::Rect`). Використовує `std::atomic<bool>` для безпечного керування життєвим циклом потоку.
*FrameProcessor*: Здійснює безпосередню обробку зображень (накладання фільтрів, зміна яскравості/насиченості у просторі HSV) та малює рамки облич, отримані з `FaceDetector`.
*KeyProcessor*: Відповідає за обробку натискань клавіш та перемикання режимів (`enum Mode`).
*Display:* Відображає фінальний результат у вікні.
