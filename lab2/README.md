# Лабораторна робота №2

## Опис завдання
Завантаження, очищення та аналіз даних VHI-індексу для областей України. Фільтрація та аналіз датасету Individual Household Electric Power Consumption.

## Інструкція
1. Створіть віртуальне середовище: `python -m venv venv`
2. Активуйте його: `venv\Scripts\activate`
3. Встановіть залежності: `pip install -r requirements.txt`
4. Запустіть Jupyter Notebook: `jupyter notebook`
