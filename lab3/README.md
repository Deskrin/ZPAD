# Лабораторна робота №3

## Опис завдання
Візуалізація датасету Auto MPG

## Вимоги
Python 3.13+

## Інструкція
1. Створіть віртуальне середовище: `python -m venv myvenv`
2. Активуйте його: `venv\Scripts\activate`
3. Встановіть залежності: `pip install -r requirements.txt`
4. Запустіть Jupyter Notebook: `jupyter notebook`
5. Запустіть файл виконанної роботи, почергово виконуючи код у комірках.
