# Лабораторна робота №6: C++ openCV

##Опис
Дана програма є  C++ додатком для захоплення, обробки та відображення відео з камери в реальному часі. Вона дозволяє застосовувати різноманітні цифрові фільтри, керувати параметрами зображення за допомогою інтерактивних елементів та відстежувати продуктивність.

##Бібліотеки та інструменти:
OpenCV, CMake, компілятор GCC/G++.

##Інструкція
Крок 1. Встановлення залежностей цією послідовністю команд:
chmod +x preinstall.sh
./preinstall.sh

Крок 2. Компіляція:
chmod +x build.sh
./build.sh

Крок 3. Запуск програми:
chmod +x run.sh
./run.sh

##Керування програмою
Гарячі клавіші:
1 - Нормальний режим (без фільтрів)
2 - Інверсія кольорів (інверсія)
3 - Gaussian Blur (розмиття)
4 - Canny фільтр (виділення меж)
5 - Ефект Сепії
6 - Фільтр Собеля
Esc - Вихід з програми

##Інтерактивні елементи:
Слайдер Яскравість: Зміна яскравості кадру
Слайдер Насиченість: Регулювання насиченості кольорів
FPS: Відображення поточної частоти кадрів
